Auto Scheduler

Auto Scheduler is a web application created as our final project in EECS 493: User Interface Development. It was created using the React and Bootstrap. Our group consisted of three members.

Auto Scheduler allows a user to input a list of tasks they'd like to complete, and it automatically organizes and times them, inserting breaks and notifying the user when it's time to switch to the next task. Auto Scheduler also saves the user's state using their web browser's local storage.
